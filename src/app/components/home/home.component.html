Modos de Edición

CQL Teacher tiene 3 modos de edición, seleccione el modo deseado.

Modo Principiante

Este modo cuenta con las siguientes sentencias:
  • Select
  • Insert
  • Update
  • Deete

Modo Intermedio

Este modo cuenta con las sentencias de principiante y además:
  • Definición de variables
  • Asignaciones
  • If/Else
  • Switch
  • While
  • Do While
  • For
  • Llamada a funciones y procedimientos

Modo Avanzado

Este modo cuenta con un editor de texto y las siguietnes opciones:
  • Abrir Archivo
  • Cerrar Archivo
  • Ejecutar
  • Editar Archivo
  • Reportes
